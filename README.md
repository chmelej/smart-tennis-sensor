# smart-tennis-sensor

Mam bluetooth sensor na tenisovou raketu. Sensor zda se funguje, ale originalni
cinska aplikace ne! take puvodni web je jiz nedostupny. cili nezbyva nic jineho
nez si aplikaci rozchodit bez cinanu. 

Puvodni aplikace je orig/Smart_Tennis_3.4.2_APKPure.apk jeji rozbalena varianta orig/Smart_Tennis.unziped/

Predstava je ze rozebereme aplikaci vytahneme jak funguje komunikace se
senzorem a napiseme vlastni aplikaci, ktera nebude odesilat data do cloudu,
bude jen spibat data z cidla.
